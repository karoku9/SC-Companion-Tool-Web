# Roadmap Decisions

## 2026-07-23 — Lock the core path to v1.0

The pre-1.0 roadmap prioritizes the reliable mission-to-execution workflow:

1. Expanded universe data.
2. Game.log assisted intake.
3. OCR assisted intake.
4. Release hardening.
5. Core v1.0 release.

Game.log is the primary automated intake source. OCR is a secondary screenshot fallback. Both must use the existing review-first mission validation flow and must never replace an active route without explicit confirmation.

Session history, companion pairing and commodity trading are deferred until after v1.0. They remain valuable expansion features but cannot delay location resolution, assisted intake or release reliability.
